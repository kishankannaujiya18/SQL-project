# Online Bookstore SQL Project

## Project Overview

This project demonstrates SQL-based analysis for an Online Bookstore Management System using PostgreSQL.

The database contains information about books, customers, and orders. Various SQL queries are used to generate business insights such as revenue analysis, customer behavior, inventory tracking, and sales performance.

## Database Schema

### Books Table

* Book ID
* Title
* Author
* Genre
* Published Year
* Price
* Stock

### Customers Table

* Customer ID
* Name
* Email
* Phone
* City
* Country

### Orders Table

* Order ID
* Customer ID
* Book ID
* Order Date
* Quantity
* Total Amount

## SQL Concepts Used

* SELECT Statements
* WHERE Clause
* ORDER BY
* Aggregate Functions
* GROUP BY
* HAVING Clause
* INNER JOIN
* LEFT JOIN
* DISTINCT
* Revenue Analysis

## Business Problems Solved

### Basic Analysis

* Retrieve books by genre
* Find books published after a specific year
* Identify customers by country
* View monthly orders
* Calculate total stock available
* Find the most expensive books
* Retrieve high-value orders

### Advanced Analysis

* Total books sold by genre
* Average book price by genre
* Customers with multiple orders
* Most frequently ordered books
* Top-selling authors
* Revenue generation analysis
* Highest spending customers
* Remaining stock calculation

## Key Insights

* Identifies best-selling books and genres
* Tracks customer purchasing behavior
* Calculates business revenue
* Monitors inventory levels
* Supports business decision-making
